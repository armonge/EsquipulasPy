Convert Decimal to a money formatted string.

Parametros:
  • places (int) - required number of places after the decimal point
  • curr (string) - optional currency symbol before the sign (may be blank)
  • sep (string) - optional grouping separator (comma, period, space, or blank)
  • dp (string) - decimal point indicator (comma or period) only specify as blank when places is zero
  • pos (string) - optional sign for positive numbers: '+', space or blank
  • neg (string) - optional sign for negative numbers: '-', '(', space or blank
  • trailneg (string) - optional trailing minus indicator: '-', ')', space or blank
Retorna: string 
>>> d = Decimal('-1234567.8901')
>>> moneyfmt(d, curr='$')
'-$1,234,567.8901'
>>> moneyfmt(d, places=0, sep='.', dp='', neg='', trailneg='-')
'1.234.568-'
>>> moneyfmt(d, curr='$', neg='(', trailneg=')')
'($1,234,567.89)'
>>> moneyfmt(Decimal(123456789), sep=' ')
'123 456 789.00'
>>> moneyfmt(Decimal('-0.02'), neg='<', trailneg='>')
'<0.0200>'
